Optimal Adaptive Cancer Therapy in a Stochastic Setting Examples

These examples are from the following paper:

View manuscript on Arxiv »

All of the computations use the following parameters:

  • Benefit per unit of acidification: \(b_a = 2.5\)
  • Benefit from the oxygen per unit of vascularization: \(b_v = 2\)
  • Cost of production VEGF: \(c = 1\)
  • Number of cells in the interaction group: \((n+1) = 5\)
  • Maximum tolerated doses: \(d_{max} = 3\)
  • Treatment time penalty: \(\delta = 0.05\)
  • Recovery barrier: \(r_b = 0.01\)
  • Failure barrier: \(1 - f_b = 0.99\)
We provide all examples on a \(1601 \times 1601 \times 6001\) grid in \((q,p,s)\) space.

Example 1 (small diffusion):

  • \(\sigma_G = \sigma_D = \sigma_V = 0.15\)

Example 2 (large diffusion):

  • \(\sigma_G = \sigma_D = \sigma_V = 0.5\)
